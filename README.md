# tic-tac-toe-with-ai-opponent

A Python-based Tic Tac Toe game with a graphical user interface (gui) and the option to play agenst another player or AI.

## Features
- GUI built with Tkinter.
- Human vs AI gameplay.
  - 2 differnet options for AI:
      - random
      - minimax algorithm 
- Win detection and highlighting the way a win was achieved. 

## How to run
```bash
python ttt_ai_and_gui.py
